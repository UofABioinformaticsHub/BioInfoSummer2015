Inspecting Data

Once we have our data loaded into R, the first thing we should do is check that it looks right.

The toothData file contains data about the effects of Vitamin C on tooth growth in Guinea Pigs, using two delivery methods, and three dose levels.

Plotting the data

R makes visually inspecting the data super-easy, and in particular, the package ggplot2 has some powerful plotting functions. If you dont have this package already installed, you can simply install this using the following commands. This is the simple method for installing packages from the main R package repository, known as CRAN. A package, is essentially a collection of functions for a some type of analysis.

install.packages("ggplot2")

To then load the functions that ggplot2 contains, we load the entire library of functions using.

library("ggplot2")
ggplot(toothData, aes(x = dose, y = len, fill = supp)) +
  geom_boxplot()

These need re-ordering

ggplot(toothData, aes(x = len)) +
  geom_histogram()
## stat_bin: binwidth defaulted to range/30. Use 'binwidth = x' to adjust this.

ggplot(toothData, aes(x = len, fill = supp)) +
  geom_density(alpha = 0.5)

ggplot(toothData, aes(x = len, fill = supp)) +
  geom_density(alpha = 0.5) +
  facet_wrap(~dose)
